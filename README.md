# Projeto-Phoenix-Android
Agente Autônomo Multi-Funcional para Android com AccessibilityService, VOZ (TTS/STT), Câmera, Banco de Conhecimento (Programação/IA/Engenharia de Prompt) e Auto-Melhoria Contínua via Gemini Agent Mode. Sistema de múltiplos agentes especializados orquestrados sob comando do General.
